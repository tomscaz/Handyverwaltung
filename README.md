Funktionen

Smartphone anlegen: Modell, Hersteller, Betriebssystem, Preis und Lagerstatus erfassen
Eingabevalidierung: leerer Modellname und ungültige/negative Preise werden abgefangen 
Filtern: Anzeige der Smartphones nach Hersteller (oder "Alle")
Auswertung: Berechnung von Anzahl, Gesamtwert und Durchschnittspreis aller erfassten Smartphones
Löschen: einzelne Auswahl oder die gesamte Liste entfernen
